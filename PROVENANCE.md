# Provenance

This repository started as a productization copy of the public GitHub repository `BOY147258/jingjitimer`.

At the time of import, the source repository did not include a license file. That means no broad open-source or commercial reuse permission should be assumed from the original repository alone.

Recommended path before commercial launch:

1. Obtain explicit permission from the original author, or
2. Treat the current code as a reference prototype and progressively replace the implementation with independently authored jjcs code, or
3. Keep use internal/private until legal ownership and licensing are clear.

This file is intentionally included so future contributors do not mistake the current repository for a clean-room commercial codebase.
